> *React · FastAPI · Python · Google Gemini API*

A **full-stack AI-powered classroom assistant** that transforms lecture recordings into comprehensive study materials with transcriptions, summaries, flashcards, and interactive quizzes.

**What it does:**
-  Transcribes lecture audio with HuggingFace Whisper
-  Generates structured study summaries
-  Creates flashcards for quick revision
-  Generates MCQ quizzes from lectures
-  Supports lecture-based Q&A chat

**Technical highlights:**
- React frontend (51.7%) with Framer Motion animations
- FastAPI backend with LangGraph orchestration

- Google Gemini API for intelligent content generation
- jsPDF for generating downloadable study materials
